right now, i don't have any binaries you can just download, so you'll have to clone my repo to install

install Git onto your machine: click here

then use that Git tool to clone my repo

for mac and linux, you type the following into your terminal:

git clone https://gitlab.com/anagabriel/pumpkin-spice-latte.git

*hit enter*

now type the following:

cd pumpkin-spice-latte

*hit enter*

cd src

*hit enter*

make

*hit enter*

if you get error messages, then you may have skipped some steps in the installation process. i recommend reading the error message to see what exactly is missing. then, goto that section of this installation guide
